## Before You Start

This guide assumes you have a Google account and basic Python knowledge. Everything runs on Colab free tier — no GPU machine needed.

## Step 1: Collect Your Images

Quality beats quantity. 500 well-annotated diverse images outperform 5,000 poorly annotated ones.

```bash
# Extract frames from video every 2 seconds
ffmpeg -i traffic_video.mp4 -vf fps=0.5 frames/frame_%04d.jpg
```

## Step 2: Annotate with Roboflow

1. Create a project — Object Detection
2. Upload images and draw bounding boxes
3. Apply augmentations: flip horizontal, brightness 15%, blur 1px
4. Export as YOLOv8 format

```yaml

```

## Step 5: Train

```python
from ultralytics import YOLO

model = YOLO('yolov8n.pt')
results = model.train(
    data='/content/traffic.yaml',
    epochs=50,
    imgsz=640,
    batch=16,
    patience=10,
    project='/content/drive/MyDrive/netra_runs',
    name='v1',
    device=0,
)
```

## Step 6: Evaluate

```python
model = YOLO('/content/drive/MyDrive/netra_runs/v1/weights/best.pt')
metrics = model.val()
print(f"mAP50: {metrics.box.map50:.3f}")
```

Project NETRA results: mAP50 of 0.847, ambulance class specifically 0.791.

## Common Pitfalls

**Colab disconnects during long runs.** Save checkpoints to Drive and resume with model.train(resume=True).

**Class imbalance kills minority class performance.** Oversample rare classes.

**Forgetting to shuffle your dataset.** Roboflow shuffles by default — verify this.
